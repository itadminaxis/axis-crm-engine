# CASAYA — Institucional

Landing page corporativa que integra el ecosistema inmobiliario digital de CASAYA. Esta página sirve como punto de entrada (Top of Funnel) para desarrolladores, compradores y asesores.

## 🔗 Ecosistema CASAYA
Este proyecto vive junto a la **Leadmachine (Sicilia Plus)**, que funciona como el motor de conversión directa.

| Proyecto | Propósito | URL |
|---|---|---|
| **Institucional** | Marca, tesis y red de desarrollos | [casaya.homes](https://casaya.homes) |
| **Leadmachine** | Conversión de prospectos (Sicilia Plus) | [vane.casaya.homes](https://vane.casaya.homes) |

## 🛠️ Stack & Despliegue
- **Tecnologías**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla).
- **Hosting**: Netlify.
- **Dominios**: Gestionados en Namecheap/Cloudflare.

## ⚙️ Integración Simbiótica
Las páginas comparten:
1. **Identidad Visual**: Paleta de colores (Rosa, Naranja, Ámbar) y tipografía (Syne, Outfit, Space Mono).
2. **Estrategia de Leads**: La página institucional redirige a los compradores a las leadmachines específicas para el cierre de ventas.
3. **Tracking**: Estructura de DataLayer consistente para analítica unificada.

## 🚀 Despliegue Local
Abre `index.html` en tu navegador o usa un servidor simple:
```bash
python3 -m http.server 8000
```
